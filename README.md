# Citizens Cards Viewer
Небольшое приложение на **Next.js** для загрузки и просмотра данных граждан из файлов игры в формате `.citb`, `.cit` или уже подготовленного `.json`.

Проект позволяет:
- загружать файл города через интерфейс
- распаковывать `.citb -> .cit`
- парсить граждан в удобные карточки
- искать и фильтровать данные
- скрывать и показывать блоки информации
- просматривать карточки в компактной сетке

---

## Возможности

### Загрузка файлов
Поддерживаются форматы:
- `.citb`
- `.cit`
- `.json`

### Обработка данных
- `.citb` распаковывается через Brotli
- при декомпрессии отбрасываются последние 4 байта
- `.cit` парсится в структурированный JSON
- данные граждан приводятся к карточкам с нормализованными полями

### Карточки граждан
Карточка содержит:
- имя
- личные данные
- волосы / глаза
- домашний адрес
- работу
- связи
- security / extra поля

Часть информации отображается компактно, часть — через tooltip или скрываемые секции.

### Поиск и фильтры
- быстрый поиск по выбранному типу поля
- фильтры по:
  - полу
  - категории роста
  - точному росту
  - точному весу
- advanced filters через добавление строк `+ Add filter`

### Управление видимостью
Можно скрывать и показывать блоки:
- Home
- Work
- Relations
- Security
- Extra

Если блок скрыт, но часть информации всё равно участвует в основной карточке, вместо значения может отображаться `***`.
